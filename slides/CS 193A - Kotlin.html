Kotlin Programming

What is Kotlin?

  • a statically typed programming language that runs on the Java VM
  • sponsored and developed by JetBrains (makers of IntelliJ IDEA)
  • can interoperate with Java code and libraries
  • As of Android Studio 3.0, the primary Android development language
  • kotlinlang.org

Printing


// Java
System.out.print("Hello");
System.out.println("Hello, world!");


// Kotlin
print("Hello")
println("Hello, world!")

Variables


// Java
int i = 10;
i += 20;

final double d = 4.50;   // constant


// Kotlin
var i = 10       // var = mutable
var i: Int = 10   // optional type
i += 20

val d = 4.50     // val = constant

Nullness (docs) 


// Java
String name = null;
int length = name.length();        // crashes at runtime :-(
if (name != null) {
	int length = name.length();    // ok
}


// Kotlin
val bad: String = null             // compiler error! can't be null
val name: String? = null           // ok
val length = name.length()         // compiler error!
val length = name?.length() ?: 0   // ok
if (name != null) {
	val length = name.length()     // ok
}

Nullness, cont'd


// Java
BankAccount acct = bank.maybeGetAccount();   // could be null
acct.deposit(5.00);                          // could crash


// Kotlin
val acct: BankAccount? = bank.maybeGetAccount()   // could be null
acct.deposit(5.00)                                // compiler error!
acct?.deposit(5.00)                               // ok
if (acct != null) {
	acct.deposit(5.00)                            // ok
}

Functions - Kotlin (docs)

}
...
val result = foo("Hello World!")
val result = foo(s: "Hello World!", n: 3)   // optional/named parameters

Classes - Java 


// Java
public class Employee {
	private String name;
	private int years;
	
	// constructor
	public Employee(String name) {
		this.name = name;
		this.years = 0;
	}
	
	// methods
	public String getName() { return name; }
	public int getYears() { return years; }
	public void setYears(int years) { this.years = years; }
	public void work() { System.out.println("Working hard!"); }
}
...
Employee ed = new Employee("Ed");
System.out.println("his name is " + ed.getName());   // getter method access
ed.setYears(5);                                      // setter method
ed.work();

Classes - Kotlin (docs) 


// Kotlin
class Employee(val name: String) {
	var years: Int = 0
	
	// methods - get/set are autogenerated (val/var)
	fun work() { println("Working hard!") }
}
...
val ed = Employee("Ed")
println("his name is " + ed.name)   // property access
ed.years = 5                        // property setter
ed.work()
